# Basic

## Tags

### ~~ASP~~

```php
<%
    // your code here
%>
```

### ~~short~~

```php
<?
    // your code here
?>
```

### script

```php
<script language="php">
    // your code here
</script>
```

### standard

```php
<?php

    // your code here

?>
```

The `ASP` and `short` are no longer supported. Use the `script` or `standard`.

The `standard` is most used.

## Comment

### single line

```php
    // your comment here
    #  another comment way like python
```

### multi line

```php
    // just like C language
    /*
     * @Author: AimerNeige
     * @Time:   2020/08/25 - 23:47
     */
```

## End of statement

Like C language, use `;`.
